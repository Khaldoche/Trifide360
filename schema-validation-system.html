5. Système de Validation - Contrôles et Qualité

Vue d'Ensemble du Système de Validation

Le système de validation de Trifide 360 assure l'intégrité et la qualité des données géospatiales à travers des contrôles multicouches. Il combine validation de schéma, règles métier, contrôles topologiques et vérifications de cohérence pour garantir des données fiables.

Types de Validation et Niveaux

ValidationLevel : enum

  • None - Pas de validation
  • Basic - Types et contraintes simples
  • Standard - Validation complète
  • Strict - Validation stricte + règles métier
  • Topology - Validation topologique
  • Custom - Validation personnalisée

ValidationScope : enum

  • Field - Validation champ individuel
  • Record - Validation enregistrement
  • Table - Validation table complète
  • Schema - Validation schéma
  • Topology - Relations topologiques
  • CrossTable - Inter-tables

ErrorSeverity : enum

  • Info - Information
  • Warning - Avertissement
  • Error - Erreur critique
  • Critical - Erreur bloquante
  • Fatal - Erreur fatale

CorrectionAction : enum

  • None - Pas de correction
  • AutoFix - Correction automatique
  • Suggest - Suggestion correction
  • Manual - Correction manuelle
  • Skip - Ignorer erreur
